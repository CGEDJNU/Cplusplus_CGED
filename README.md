# Cplusplus_CGED

中文语法检纠错项目，应用于文本自动生成后的自动检纠（大大解放我司编辑妹子们的生产力），用于HSK的作文评分等场景。


原始代码来自@DancingSoul,感谢作者分享。本项目分只享改进的关键代码，数据和分词预训练模型较大，暂不上传，项目近期持续更新中。
